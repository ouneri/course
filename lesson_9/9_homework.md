# Домашнее задание 9: Введение в Angular для Quiz App

## Цель

Создать первое Angular-приложение для управления квизами, ознакомиться с основными концепциями фреймворка и реализовать базовые задачи веб-приложения. Это задание укрепит понимание компонентной архитектуры, сервисов, форм и реактивного программирования.

## Задания

### Angular-приложение

Создать Angular приложение через CLI.

### Модели

Описать интерфейсы и классы для квизов.

### Сервисы

Создать абстракции дли работы с квизами, реализовать мок-сервисы.

### Страницы

Треубется реализовать страницу со списком квизов с пагинацией. Требуется реализовать страницы создания, редактирования, удаления квизов.
