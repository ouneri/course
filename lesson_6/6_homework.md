# Домашнее задание 6: JavaScript - добавляем интерактивность портфолио

## Цель

Освоить основы JavaScript и научиться добавлять интерактивность на веб-страницы. Вы превратите свою статическую HTML страницу в динамический и отзывчивый сайт с пользовательскими интерактивными элементами.

## Что нужно сделать

### Настройка окружения

В вашем Vite проекте создайте файл `script.js` в папке `js` и подключите его к `index.html`. Убедитесь, что JavaScript выполняется без ошибок и консоль браузера чистая.

### Интерактивная навигация

Реализуйте плавную прокрутку к секциям при клике на пункты меню. При прокрутке страницы подсвечивайте текущий раздел в навигации. Создайте мобильное меню (бургер-меню) с анимацией открытия/закрытия для экранов меньше 768px.

### Валидация формы контактов

Добавьте JavaScript валидацию для формы "Предложить сотрудничество". Проверяйте корректность email адреса, обязательность полей, минимальную длину сообщения. Показывайте пользователю понятные сообщения об ошибках и успешной отправке.

### Динамические элементы

Создайте несколько динамических элементов на странице: таймер обратного отсчета до события, динамическое отображение текущего времени и даты.

### Темная тема

Реализуйте переключатель темы (светлая/темная) с сохранением выбора в localStorage. Создайте CSS переменные для цветов темы и применяйте их динамически. Учитывайте системные настройки пользователя (`prefers-color-scheme`).

## Критерии выполнения

- [ ] JavaScript подключен и работает без ошибок в консоли
- [ ] Плавная прокрутка к секциям при клике на навигацию
- [ ] Подсветка активного раздела в навигации при прокрутке
- [ ] Мобильное меню с анимацией открытия/закрытия
- [ ] Валидация формы контактов с понятными сообщениями об ошибках
- [ ] Переключение светлой/темной темы с сохранением в localStorage
- [ ] Адаптивность на мобильных устройствах
