# Домашнее задание 11: Auth

## Цель

Разобраться в аутентификации, реализовать интеграцию с реальным API.

## Задания

### Развернуть бэкенд

В корне репозитория есть [директория с проектом](./../server/CMQuiz). Требуется ИЛИ запустить приложение через docker, ИЛИ установить dotnet и запустить приложение на своём компьютере.

### Реализовать интеграцию

#### API

Требуется подключить HTTP клиент в angular, подменить mock service, который был реалзиован Вами ранее, на обращение к серверу.

#### Auth

Требуется реализовать страницы регистрации и аутентификации, которые будут обращаться к auth методам сервера для получения куки.
