# Домашнее задание 10: ReactiveFormsModule

## Цель

Разобраться в ReactiveFormsModule в Angular. Реализовать создание форм с несколькими типами вопросов.

## Задания

### ReactiveFormsModule

Подключить ReactiveFormsModule в компонент создания квизов (или в app.config.ts).

### Реализовать создание нескольких типов вопросов

Требуется реализовать создание нескольких вопросов:

- [ ] Select (Dropdown)
- [ ] Select (Multiselect)
- [ ] Text
- [ ] Range
- [ ] DateTime
