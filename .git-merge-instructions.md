# Инструкция по получению изменений из репозитория наставника

## Шаги для получения изменений:

1. **Добавьте удаленный репозиторий наставника как `upstream`:**
   ```bash
   git remote add upstream <URL_репозитория_наставника>
   ```
   Например: `git remote add upstream https://github.com/mentor-username/course-main.git`

2. **Если у вас уже есть ваш собственный форк на GitHub, добавьте его как `origin`:**
   ```bash
   git remote add origin <URL_вашего_форка>
   ```

3. **Получите все изменения из upstream:**
   ```bash
   git fetch upstream
   ```

4. **Переключитесь на основную ветку (обычно main или master):**
   ```bash
   git checkout -b main
   ```
   Или если ветка уже существует:
   ```bash
   git checkout main
   ```

5. **Слейте изменения из upstream в вашу локальную ветку:**
   ```bash
   git merge upstream/main
   ```
   (замените `main` на `master` если у наставника используется master)

## Если возникнут конфликты при слиянии:
- Git покажет конфликтующие файлы
- Отредактируйте их, разрешив конфликты
- Затем выполните:
  ```bash
  git add .
  git commit -m "Merge upstream changes"
  ```
